Welcome to Korita

Korita is an on-line data discretization tool, built on the pydida discretization package. It can also be used as a simple format conversion tool.

Korita consists of five simple steps:

  1. Data upload (this page)
  2. Data format specification
  3. Discretization commands specification
  4. Output format specification
  5. Data download

Data upload

Korita expects the data be in some sort of tabular ASCII format. The specific input format will be defined in step 2. First, however, the data has to be uploaded to the server.
